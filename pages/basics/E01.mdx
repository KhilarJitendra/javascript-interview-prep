## E01

### Variables in JavaScript

JavaScript variables can be declared in four ways:

- Automatically
- Using \`var\`
- Using \`let\`
- Using \`const\`

#### Automatically Declared Variables

Variables are implicitly declared without using \`var\`, \`let\`, or \`const\`. This is not recommended for clarity and may lead to unintended global variables.

#### Using `var`

Declares variables with function-level scope and has quirks like hoisting.

```javascript
function example() {
  if (true) {
    var x = 10;  // Function-scoped
  }
  console.log(x);  // 10
}

example();

```

### Using `let`

In JavaScript, the `let` keyword is used to declare variables with block-level scope. This is particularly useful for loop counters and limiting variable visibility.

```javascript
function loopExample() {
  for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4
  }
  // console.log(i);  // Error: i is not defined outside the loop
}

```

## Using `const` in JavaScript

In JavaScript, the `const` keyword is used to declare constant variables. Constants are values that should not be reassigned, enhancing code readability and preventing accidental modifications.

```javascript
const PI = 3.14;
const DAYS_IN_WEEK = 7;

// Attempting to reassign a constant will result in an error
// PI = 3.14159;  // Error: Assignment to constant variable

```

## Best Use Cases for Variable Declaration in JavaScript

### Automatic Declaration:

- Generally not recommended for clear code.
- Use explicit declarations (\`var\`, \`let\`, \`const\`) for better practices.

### Using \`var\`:

- Suitable for variables with function-level scope.
- Be mindful of quirks like hoisting.

### Using \`let\`:

- Appropriate for variables with block-level scope.
- Useful for loop counters and limiting visibility.

### Using \`const\`:

- Ideal for constant variables that should not be reassigned.
- Enhances code readability and prevents accidental modifications.

```
// Using var
function demonstrateVar() {
  if (true) {
    var x = 10;
    console.log("Inside if block (using var):", x);
  }

  console.log("Outside if block (using var):", x);
}

// Using let
function demonstrateLet() {
  if (true) {
    let y = 20;
    console.log("Inside if block (using let):", y);
  }

  // Uncommenting the line below would result in an error because 'y' is not defined outside the block.
  // console.log("Outside if block (using let):", y);
}

// Using const
function demonstrateConst() {
  const z = 30;
  console.log("Using const:", z);

  // Uncommenting the line below would result in an error because you cannot reassign a value to a const variable.
  // z = 40;
}

// Call the functions to see the output
demonstrateVar();
demonstrateLet();
demonstrateConst();

```
