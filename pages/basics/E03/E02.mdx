## E02

### JavaScript String Methods:

1. **String length:**
   - Returns the length of a string.

2. **String slice():**
   - Extracts a portion of a string and returns a new string.

3. **String substring():**
   - Similar to slice but cannot accept negative indexes.

4. **String substr():**
   - Similar to slice but the second parameter specifies the length of the extracted part.

5. **String replace():**
   - Searches a string for a specified value and replaces the first occurrence.

6. **String replaceAll():**
   - Searches a string for a specified value and replaces all occurrences.

7. **String toUpperCase():**
   - Converts a string to uppercase.

8. **String toLowerCase():**
   - Converts a string to lowercase.

9. **String concat():**
   - Concatenates two or more strings.

10. **String trim():**
    - Removes whitespace from both ends of a string.

11. **String trimStart():**
    - Removes whitespace from the beginning of a string.

12. **String trimEnd():**
    - Removes whitespace from the end of a string.

13. **String padStart():**
    - Pads a string with another string until the resulting string reaches a specified length.

14. **String padEnd():**
    - Pads a string with another string until the resulting string reaches a specified length.

15. **String charAt():**
    - Returns the character at a specified index in a string.

16. **String charCodeAt():**
    - Returns the Unicode of the character at a specified index in a string.

17. **String split():**
    - Splits a string into an array of substrings based on a specified separator.


### JavaScript String Search Methods:

1. **String indexOf():**
   - Returns the index of the first occurrence of a specified value in a string. Returns -1 if the value is not found.

2. **String lastIndexOf():**
   - Returns the index of the last occurrence of a specified value in a string. Returns -1 if the value is not found.

3. **String search():**
   - Searches a string for a specified value and returns the position of the match. It can use regular expressions.

4. **String match():**
   - Searches a string for a specified value or a regular expression and returns an array of the matches.

5. **String matchAll():**
   - Returns an iterator of all the matches of a pattern in a string. It uses regular expressions.

6. **String includes():**
   - Determines whether a string contains a specified value. Returns `true` if found, `false` otherwise.

7. **String startsWith():**
   - Checks if a string starts with a specified sequence of characters. Returns `true` if the string starts with the specified characters, `false` otherwise.

8. **String endsWith():**
   - Checks if a string ends with a specified sequence of characters. Returns `true` if the string ends with the specified characters, `false` otherwise.


N.B :

Did You Notice?
The two methods, indexOf() and search(), are equal?

They accept the same arguments (parameters), and return the same value?

The two methods are NOT equal. These are the differences:

The search() method cannot take a second start position argument.
The indexOf() method cannot take powerful search values (regular expressions).
